# Demo Script

1. Open the dashboard at `/live`.
2. Replay the scenario.
3. Safe test and pull request actions are allowed.
4. Main branch merge requires approval.
5. Production database migration is forced through dry-run evidence.
6. Production deployment requires approval.
7. Approved execution is performed through AgentGate with persisted logs.
8. Audit trace shows request, decision, approval, token, execution, and result.

Phase 0 only creates the fixtures and placeholder surfaces for this flow.
